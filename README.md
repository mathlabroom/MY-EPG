## 用法
MY-EPG.channels.xml、MY-EPG.sources.xml需要放在你的enigma2机顶盒的/etc/epgimport/目录下，MY-EPG.channels.xml中的参考要修改成你机器内频道的参考，在epg import中选择MY-EPG源，导入，enjoy！
